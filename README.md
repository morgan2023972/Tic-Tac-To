# Tic‑Tac‑Toe (Morpion) — CLI

![CI](https://github.com/<TON-USER>/<TON-REPO>/actions/workflows/ci.yml/badge.svg)

Jeu simple de morpion en ligne de commande (Node.js).

Prérequis
- Node.js 18+

Installation et exécution
1. Ouvrez un terminal dans le dossier du projet.
2. Lancez :

```
npm start
```

Le jeu est pour deux joueurs humains en alternance. Entrez un nombre de 1 à 9 pour jouer dans la case correspondante.
